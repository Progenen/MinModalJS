ModalsJS

Description:
MinModalJS - is a lightweight and easy-to-use library for creating unlimited modal windows

Features:
1. Written in vanilla javascript, no dependencies needed
2. Lightweight JS+CSS(minified: ~1.22kb)
3. No restrictions on the number of buttons and modal windows
4. Modal closed if you press ESC(you can replace event.code or off it) or click in outside modal (you can off it). Default all features turn on
5. MinModalJS haves three methods modalOpen(), modalDestroy() and modalClose()
6. You can see the code of any key on the site https://keycode.info/ you need a event.code
